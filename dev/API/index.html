BioMakie.getplottingdata — Method
getplottingdata( msa )::Tuple{Matrix{String}, Vector{String}, Vector{String}}

Collects data for plotting (residue string matrix, x labels, and y labels).

This function converts an AbstractMultipleSequenceAlignment (from MIToS.MSA), or a Vector{Tuple{String,String}} (from FastaIO), or a Vector{FASTX.FASTA.Record} to a matrix of residue characters, x labels, and y labels.

BioMakie.msavalues — Function
msavalues( msa::AbstractMatrix, resdict::AbstractDict )::Matrix{Real}

Returns a matrix of numbers according to the given dictionary, where keys are residue letters and values are numbers. This matrix is used as input for plotmsa for the heatmap colors.

Default values for residue letters are from Kidera Factor values. kf 2 is Kidera Factor 2 (size/volume-related). The KF dictionary is in utils.jl.

BioMakie.plotstruc! — Method
plotstruc!( fig, structure )

Plot a protein structure into a Figure. Position can be specified with the gridposition keyword argument.

Examples

fig = Figure()
using BioStructures
struc = retrievepdb("2vb1", dir = "data/") |> Observable
sv = plotstruc!(fig, struc)

struc = read("data/2vb1_mutant1.pdb", BioStructures.PDB) |> Observable
sv = plotstruc!(fig, struc)

Optional Arguments:

  • selectors ––- [standardselector]
  • resolution –– (800,800)
  • gridposition – (1,1)
  • plottype ––– :ballandstick, another option is :spacefilling
  • atomcolors –– elecolors, another option is aquacolors, or define your own dict for atoms like: "N" => :blue

BioMakie.plotstruc — Method
plotstruc(structure)

Create and return a Makie Figure for a protein structure, along with the structure wrapped in an Observable if it wasn't Observable already.

Examples

using BioStructures
struc = retrievepdb("2vb1", dir = "data/") |> Observable
sv = plotstruc(struc)

struc = read("data/2vb1_mutant1.pdb", BioStructures.PDB) |> Observable
sv = plotstruc(struc)

Optional Arguments:

  • selectors ––- [standardselector]
  • resolution –– (800,800)
  • gridposition – (1,1)
  • plottype ––– :ballandstick, another option is :spacefilling
  • atomcolors –– elecolors, another option is aquacolors, or define your own dict for atoms like: "N" => :blue
